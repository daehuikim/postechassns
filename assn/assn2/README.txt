This assignment can be done individually in groups of two. It is worth a total of 4% of the final mark.

 

The assignment is to develop an Internet text chatting program using sockets. 

(Recommended programming language: Python, 
but C/C++/Java also possible)

 

 You are to submit the following:


1. Design and Implementation Document (5 or less pages)
2. Source code (well-documented)
3. Executable codes
4. Screen capture video of the program executions

 

// 보고서에 Source Code 부분을 첨부하였고 3.Executable codes로 server.py와 client.py를 제출합니다. 위 두 파일은 python3.0이상으로 컴파일하면 됩니다.
Screen capture video는 ip주소를 알아내어 들어가는 부분과 client가 접속하고 종료하는 것까지 녹화하였습니다.
